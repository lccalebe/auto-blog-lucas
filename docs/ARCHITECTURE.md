### System Architecture
```
┌─────────────────────────────────────────────────────────┐
│                    GitHub Repository                    │
│                  (Source Code + CI/CD)                  │
└────────────────────┬────────────────────────────────────┘
                     │ git push
                     ↓
┌─────────────────────────────────────────────────────────┐
│                   AWS CodeBuild                         │
│  • Builds Docker images                                 │
│  • Pushes to ECR                                        │
│  • Auto-deploys to EC2                                  │
└────────────────────┬────────────────────────────────────┘
                     │
                     ↓
┌─────────────────────────────────────────────────────────┐
│              AWS Elastic Container Registry             │
│        (Stores Docker Images: Frontend + Backend)       │
└────────────────────┬────────────────────────────────────┘
                     │
                     ↓
┌─────────────────────────────────────────────────────────┐
│                      AWS EC2 Instance                   │
│  ┌──────────────────────────────────────────────────┐   │
│  │  Docker Compose Orchestration                    │   │
│  │                                                  │   │
│  │  ┌──────────────┐  ┌──────────────┐  ┌─────────┐ │   │
│  │  │   Frontend   │  │   Backend    │  │ Postgres│ │   │
│  │  │   (Nginx)    │  │  (Node.js)   │  │   DB    │ │   │
│  │  │   Port 80    │  │   Port 5000  │  │         │ │   │
│  │  └──────────────┘  └──────────────┘  └─────────┘ │   │
│  │                           │                      │   │
│  │                           ↓                      │   │
│  │                  ┌────────────────┐              │   │
│  │                  │ Cron Scheduler │              │   │
│  │                  │ (Every 3Hours) │              │   │
│  │                  └────────┬───────┘              │   │
│  └───────────────────────────┼──────────────────────┘   │
└────────────────────────────┬─┼──────────────────────────┘
                             │ │
                             ↓ ↓
                    ┌─────────────────┐
                    │ HuggingFace API │
                    │  (AI Content)   │
                    └─────────────────┘
```
